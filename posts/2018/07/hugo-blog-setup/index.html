Hugo Blog Setup

July 16, 2018 #note #hugo #blog

Blog 這件事大概是架了十個一篇文章都沒寫的狀態，忙碌和頹廢的生活哪能提起抒發心志的動力呢？ 身為一位軟體工程師偶爾想要記錄一下自己所踩過的地雷，寫下提醒給未來的自己別栽在同一個 bug 手上。

Static Web Site Generators 目前有很多選擇 [1, 2]， 也用過其他 framework 像是 Hexo，至於為什麼會選擇 Hugo 原因大概是想學點 Go(Golang)， 拜 Go 為系統語言所賜速度也算快，且沒有相依到其他軟體（如果使用 binary release ）。

至於要怎麽架設也非常容易，照著官網的文件 [3]，可以很容易地架起 local server， theme 的選擇也非常多，官方也整理的不錯 [4]， 不同 theme 能做不同用途（像是 project official site），但列表還是個人 blog 居多，從其他 framework port 過來的也很多， 尤其是 Jekyll。

調整好 blog 的設定然後寫篇短文，把 public/ 都丟到網路上就可以了，因為是單純的靜態網頁所以放到各種 web server 都是可以的。 GitHub pages 是普羅大眾最常用的， 進階一點搭配 custom domain 和 CDN 就完整了 [5]。
